YouTube Video Downloader

A Flask application to download and organize YouTube videos.

Features

  • Download YouTube videos directly to the server
  • Organize videos in a library for easy access
  • Play videos directly in the browser
  • Track metadata including duration and thumbnail

Setup Instructions

  1. Clone this repository
  2. Install dependencies: pip install Flask yt-dlp
  3. Run the server: python server.py
  4. Open http://localhost:5000 in your browser
